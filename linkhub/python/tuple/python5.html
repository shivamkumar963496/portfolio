# Question 5:
# Ek Python program likho jo tuple slicing karke different parts print kare.

# tuple create karna
t = (10, 20, 30, 40, 50)

# index 1 se 3 tak slice karna
print(t[1:4])

# starting se index 2 tak slice karna
print(t[:3])

# index 2 se end tak slice karna
print(t[2:])
📊 Output:

(20, 30, 40)
(10, 20, 30)
(30, 40, 50)
