# Question 17:
# Replace all vowels in a string with '*'.

# input from user
s = input("enter a sentence : ")

# initialize result
result = " "

# loop through string
for ch in s:
    if ch in "aeiouAEIOU":
        result += "*"
    else:
        result += ch

# print result
print(result)
📊 Output (Example):

Input: Shivam
Output: Sh*v*m
