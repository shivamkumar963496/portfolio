# Question 13:
# Dictionary me min value aur min key find karo

# dictionary create karna
d = {"a": 5, "b": 50}

# min value nikalna
print("dictionary ki sabse choti value :", min(d.values()))

# min key nikalna
print("dictionary ki sabse choti key :", min(d.keys()))
📊 Output:

dictionary ki sabse choti value : 5
dictionary ki sabse choti key : a
