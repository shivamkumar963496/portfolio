# Question 13:
# Print Fibonacci series up to n terms.

# input lena
n = int(input("Enter a number: "))

# starting values
a = 0
b = 1
i = 1

# n terms tak print
while i <= n:
    print(a, end=" ")
    c = a + b   # next term
    a = b
    b = c
    i += 1
📊 Output:

Enter a number: 6
0 1 1 2 3 5
