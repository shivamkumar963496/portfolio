# Ek Python program likho jo tuple me kisi element ki frequency count kare.

# tuple create karna
t = (22, 36, 98, 75, 49, 86, 91, 43, 25, 77, 89, 22, 35, 98, 78, 86, 22)

# element ki count nikalna
print(f"Tuple me {22} kitni baar aaya hai :", t.count(22))

Tuple me 22 kitni baar aaya hai : 3
