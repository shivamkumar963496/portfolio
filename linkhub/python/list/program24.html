# Question 24:
# 1 se n tak ke even numbers ki list banani hai

# User se range input li
n = int(input("ENTER RANGE : "))

# List comprehension se even numbers nikale
even_list = [i for i in range(1, n) if i % 2 == 0]

# Output print kiya
print(f"List of even numbers from 1 to {n} :", even_list)

Input: 10
