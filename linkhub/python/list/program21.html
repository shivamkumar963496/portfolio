# Question 21:
# List me se even numbers nikal kar unki count find karni hai

# List create ki
nums = [10,25,5,40,12,36,98,75,45,68,92,91]

# List comprehension se even numbers nikale
even_list = [i for i in nums if i % 2 == 0]

# Total count nikali
total = len(even_list)

# Output print kiya
print("Even Numbers:", even_list)
print("Total Even Count:", total)
📊 Output:

Even Numbers : [10, 40, 12, 36, 98, 68, 92]
Total Even Count : 7
