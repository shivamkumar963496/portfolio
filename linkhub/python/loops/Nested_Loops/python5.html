# Question 5:
# Print numbers using nested loop and stop when number becomes 3 (break).

# outer loop for rows
for i in range(3):

    # inner loop for numbers
    for j in range(1,5):

        # stop loop when j == 3
        if j == 3:
            break

        print(j, end=" ")
        
    # move to next line
    print()
📊 Output:

1 2
1 2
1 2
