# Question 11:
# Print hollow square star pattern using nested loop.

# size of square
n = 5

# outer loop
for i in range(n):

    # inner loop
    for j in range(n):

        # print border stars
        if i == 0 or i == n-1 or j == 0 or j == n-1:
            print("*", end=" ")
        else:
            print(" ", end=" ")

    # move to next line
    print()
📊 Output:

* * * * *
* *
* *
* *
* * * * *
