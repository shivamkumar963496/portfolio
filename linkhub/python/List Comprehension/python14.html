# Question 14:
# List comprehension ka use karke
# negative aur positive numbers identify karo.

# Number list create karna
nums = [-2, 5, -1, 7]

# Negative aur positive check karna
result = ["Negative" if x < 0 else "Positive" for x in nums]

# Final output print karna
print(result)
📊 Output:

['Negative', 'Positive', 'Negative', 'Positive']
