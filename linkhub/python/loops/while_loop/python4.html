# Question 4:
# Print odd numbers from 1 to n.

# input lena
n = int(input("Enter number: "))

# start 1 se
i = 1

# loop 1 se n tak chalega
while i <= n:
    
    # odd check
    if i % 2 != 0:
        print(i, end=" ")
    
    # next number
    i += 1
📊 Output:

Enter number: 10
1 3 5 7 9
