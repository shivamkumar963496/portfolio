# Question 11:
# Write a Python program to take a number as input
# and print its reverse.

# user se number input liya ja raha hai
num = int(input("ENTER A NUMBER : "))

# reverse store karne ke liye variable
rev = 0

# jab tak number 0 se bada hai tab tak loop chalega
while num > 0:
    digit = num % 10
    rev = rev * 10 + digit
    num = num // 10

# final reverse number print kiya
print("REVERSE NUMBER :", rev)
