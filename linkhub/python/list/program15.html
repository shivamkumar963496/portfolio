# Question 15:
# List ko reverse karna hai (loop ka use karke)

# List create ki
nums = [10,25,5,40,12,36,98,75,45,68,92,91]

# Empty list banayi reverse ke liye
rev = []

# Loop chala kar reverse list banayi
for i in nums:
    rev = [i] + rev   # element ko aage insert kar rahe hain

# Output print kiya
print(rev)
📊 Output:

[91, 92, 68, 45, 75, 98, 36, 12, 40, 5, 25, 10]
