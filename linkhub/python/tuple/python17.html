# Question 17:
# Ek Python program likho jo list ko tuple me convert kare aur unka type print kare.

# list create karna
t = [22, 36, 98, 75, 49, 86, 91, 43, 25, 77, 89)

# list print karna
print("\n", t)

# list ka type check karna
print("\n", type(t))

# list ko tuple me convert karna
result = tuple(t)

# tuple print karna
print("\n", result)

# tuple ka type check karna
print("\n", type(result))
📊 Output:

[22, 36, 98, 75, 49, 86, 91, 43, 25, 77, 89]
<class 'list'>
(22, 36, 98, 75, 49, 86, 91, 43, 25, 77, 89)
<class 'tuple'>
