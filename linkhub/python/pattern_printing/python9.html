# Question 9:
# Print inverted right aligned pyramid star pattern using nested loop.

# outer loop (reverse)
for i in range(5, 0, -1):

    # space loop
    for j in range(5 - i):
        print(" ", end="")

    # star loop
    for k in range(i):
        print("*", end=" ")

    # move to next line
    print()
📊 Output:

* * * * *
* * * *
* * *
* *
*
