# Question 25:
# List me even numbers ki index positions find karni hai

# List create ki
nums = [10,25,5,40,12,36,98,75,45,68,92,91,5,25,10,98]

# List comprehension se even elements ki index nikali
even_index = [i for i in range(len(nums)) if nums[i] % 2 == 0]

# Output print kiya
print("Even number ki index values:", even_index)
📊 Output:

Even number ki index values : [0, 3, 4, 5, 6, 9, 10, 14, 15]
