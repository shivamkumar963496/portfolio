# Question 22:
# Ek Python program likho jo tuple se duplicate elements hata kar unique tuple banaye.

# tuple create karna
t = (22, 36, 98, 75, 49, 86, 91, 43, 25, 77, 89, 22, 89, 75, 36)

# duplicates remove karna
unique_tuple = tuple(set(t))

# output print karna
print("Unique tuple (no duplicate elements):", unique_tuple)
📊 Output:

Unique tuple (no duplicate elements): (98, 36, 75, 43, 77, 49, 86, 22, 89, 91, 25)
