# Question 7:
# Ek Python program likho jo dictionary se kisi key ko delete kare.

# dictionary create karna
d = {"a": 10, "b": 20, "c": 30}

# old dictionary print karna
print("Old dictionary :", d)

# key define karna
key = "b"

# key delete karna
del d[key]

# updated dictionary print karna
print("After delete key :", d)
📊 Output:

Old dictionary : {'a': 10, 'b': 20, 'c': 30}
After delete key : {'a': 10, 'c': 30}
