# Question 15:
# Print Pascal Triangle pattern using nested loop.

# total rows
n = 4

# outer loop
for i in range(n):

    # starting number
    num = 1

    # inner loop
    for j in range(i+1):
        print(num, end=" ")

        # calculate next value
        num = num * (i - j) // (j + 1)

    # move to next line
    print()
📊 Output:

1
1 1
1 2 1
1 3 3 1
