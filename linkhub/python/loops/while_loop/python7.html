# Question 7:
# Count digits in a number.

# input lena
n = int(input("ENTER A NUMBER = "))

# count start 0 se
count = 0

# jab tak number 0 nahi hota
while n != 0:
    n //= 10   # last digit remove
    count += 1 # count increase

# result print
print("TOTAL DIGITS IN GIVEN NUMBER =", count)
📊 Output:

ENTER A NUMBER = 12345
TOTAL DIGITS IN GIVEN NUMBER = 5
