# Question 29:
# Dictionary se un items ko new dictionary me store karo
# jinki value 10 se chhoti ya barabar ho

# dictionary create karna
d = {"a": 5, "b": 15, "c": 8}

# new dictionary banana
new_dic = {}

# condition check karke filter karna
for k, v in d.items():
    if v <= 10:
        new_dic[k] = v

# output print karna
print(new_dic)
📊 Output:

{'a': 5, 'c': 8}
