# Question 22:
# Print increasing odd star pattern.

# input lena
n = int(input("Enter a number: "))

# outer loop (rows)
i = 1

while i <= n:
    j = 1
    
    # stars = 2*i - 1
    while j <= (2 * i - 1):
        print("*", end=" ")
        j += 1
        
    print()   # next line
    i += 1
📊 Output:

Enter a number: 4
*
* * *
* * * * *
* * * * * * *
