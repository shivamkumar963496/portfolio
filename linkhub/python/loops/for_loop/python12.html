# Question 12:
# Write a Python program to print Fibonacci series up to n terms.

# user se limit input liya ja raha hai
n = int(input("ENTER LIMIT : "))

# first two numbers initialize kiye
a = 0
b = 1

# loop n terms tak chalega
for i in range(n):
    print(a, end=" ")
    c = a + b
    a = b
    b = c
