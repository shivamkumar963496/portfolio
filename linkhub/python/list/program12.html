# Question 12:
# List me se duplicate hata kar second smallest element find karna hai

# List create ki
nums = [10,25,5,40,12,36,98,75,45,68,92,91,5]

# set se duplicates remove kiye aur sorted list banayi
sorted_list = sorted(set(nums))

# Sorted list print ki
print("Sorted list :", sorted_list)

# Second smallest element nikala
print("Second smallest number :", sorted_list[1])
📊 Output:

Sorted list : [5, 10, 12, 25, 36, 40, 45, 68, 75, 91, 92, 98]
Second smallest number : 10
