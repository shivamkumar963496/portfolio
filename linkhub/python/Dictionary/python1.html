# Question 1:
# Ek Python program likho jo do lists ko zip karke dictionary banaye.

# lists create karna
name = ["shivam", "yograj", "akhand", "rajveer", "atul"]
age = [23, 34, 21, 23, 54]

# dictionary banana
d = dict(zip(name, age))

# output print karna
print(d)
📊 Output:

{'shivam': 23, 'yograj': 34, 'akhand': 21, 'rajveer': 23, 'atul': 54}
