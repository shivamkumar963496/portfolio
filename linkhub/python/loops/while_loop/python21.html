# Question 21:
# Print inverted star pattern.

# input lena
n = int(input("Enter a number: "))

# outer loop (rows)
i = n

while i >= 1:
    j = 1
    
    # inner loop (stars)
    while j <= i:
        print("*", end=" ")
        j += 1
    
    print()   # next line
    i -= 1
📊 Output:

Enter a number: 5
* * * * *
* * * *
* * *
* *
*
