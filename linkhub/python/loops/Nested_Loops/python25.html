# Question 25:
# Print multiplication values but skip when i == j using continue.

# outer loop
for i in range(1,4):

    # inner loop
    for j in range(1,4):

        # skip when i == j
        if i == j:
            continue

        print(i * j, end=" ")

    # move to next line
    print()
📊 Output:

2 3
2 6
3 6
