# Question 9:
# Find sum of digits of a number.

# input lena
n = int(input("Enter a number: "))

# sum store karne ke liye
sum = 0

# jab tak number 0 nahi hota
while n != 0:
    last_digit = n % 10   # last digit nikalo
    sum += last_digit     # sum me add karo
    n //= 10              # last digit hatao

# result print
print("Sum of digits =", sum)
📊 Output:

Enter a number: 1234
Sum of digits = 10
