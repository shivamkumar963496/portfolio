# Question 21:
# List comprehension ka use karke
# 1 se 20 tak prime numbers find karo.

# Last number define karna
n = 20

# Prime numbers find karna
result = [
    x for x in range(2, n + 1)
    if all(x % y != 0 for y in range(2, x))
]

# Final output print karna
print(result)
📊 Output:

[2, 3, 5, 7, 11, 13, 17, 19]
