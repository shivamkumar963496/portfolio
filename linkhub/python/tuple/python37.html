# Question 37:
# Ek Python program likho jo do tuples ka index-wise sum nikaale.

# tuples create karna
t1 = (2, 4, 8, 9, 7, 5, 6, 4, 8, 9)
t2 = (4, 8, 5, 7, 9, 6, 2, 3, 5, 8)

# index-wise sum nikalna
result = tuple(t1[i] + t2[i] for i in range(len(t1)))

# output print karna
print("Tuples ka index-wise sum :", result)
📊 Output:

Tuples ka index-wise sum : (6, 12, 13, 16, 16, 11, 8, 7, 13, 17)
