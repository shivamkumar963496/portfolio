# Question 41:
# List of dictionaries ko ek single dictionary me merge karo

# list create karna
lst = [{"a": 1}, {"b": 2}, {"c": 3}]

# result dictionary
result = {}

# loop ke through merge karna
for d in lst:
    result.update(d)

# output print karna
print(result)
📊 Output:

{'a': 1, 'b': 2, 'c': 3}
