# Question 32:
# List me se duplicates hata kar 2nd maximum element find karna hai

# List create ki
nums = [10,25,5,40,12,36,98,75,45,68,92,91,5,25,10,98]

# set se duplicates remove kiye aur sorted list banayi
unique_sorted = sorted(set(nums))

# 2nd maximum element nikala
print(unique_sorted[-2])
📊 Output:

92
