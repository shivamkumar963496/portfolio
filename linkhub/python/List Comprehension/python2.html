# Question 2:
# List comprehension ka use karke
# list me se even numbers filter karo.

# Number list create karna
nums = [1, 2, 3, 4, 5, 6]

# Even numbers filter karna
result = [x for x in nums if x % 2 == 0]

# Final output print karna
print(result)
📊 Output:

[2, 4, 6]
