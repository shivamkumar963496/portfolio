# Question 1:
# List comprehension ka use karke
# har number ka square find karo.

# Number list create karna
nums = [1, 2, 3, 4]

# Har number ka square nikalna
result = [num * num for num in nums]

# Final output print karna
print(result)
📊 Output:

[1, 4, 9, 16]
