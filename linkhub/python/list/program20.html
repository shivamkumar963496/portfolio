# Question 20:
# List me se odd numbers nikal kar unki count find karni hai

# List create ki
nums = [10,25,5,40,12,36,98,75,45,68,92,91]

# List comprehension se odd numbers nikale
odd_list = [i for i in nums if i % 2 != 0]

# Total count nikali
total = len(odd_list)

# Output print kiya
print("Odd Numbers:", odd_list)
print("Total Odd Count:", total)
📊 Output:

Odd Numbers : [25, 5, 75, 45, 91]
Total Odd Count : 5
