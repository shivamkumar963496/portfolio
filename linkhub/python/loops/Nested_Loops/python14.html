# Question 14:
# Print (i,j) pairs and stop row when i == j using break.

# outer loop
for i in range(1,4):

    # inner loop
    for j in range(1,4):

        print(f"({i},{j})", end=" ")

        # stop row when i == j
        if i == j:
            print("→ stoprow")
            break
📊 Output:

(1,1) → stoprow
(2,1) (2,2) → stoprow
(3,1) (3,2) (3,3) → stoprow
