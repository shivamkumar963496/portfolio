# Question 29:
# List me se duplicate hata kar sorted unique list banani hai

# List create ki
nums = [10,25,5,40,12,36,98,75,45,68,92,91,5,25,10,98]

# set se duplicates remove kiye aur sorted se arrange kiya
print(sorted(set(nums)))
📊 Output:

[5, 10, 12, 25, 36, 40, 45, 68, 75, 91, 92, 98]
