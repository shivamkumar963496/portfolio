# Question 3:
# Print inverted triangle star pattern using nested loop.

# total rows
n = 6

# outer loop (reverse)
for i in range(n, 0, -1):

    # inner loop for stars
    for j in range(i):
        print("*", end=" ")
        
    # move to next line
    print()
📊 Output:

* * * * * *
* * * * *
* * * *
* * *
* *
*
