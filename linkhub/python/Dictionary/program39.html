# Question 39:
# Dictionary ke items ko value ke basis par descending order me sort karo
# aur top 3 items print karo

# dictionary create karna
d = {"a": 10, "b": 50, "c": 30, "d": 40}

# sorting karna (value ke basis par)
sorted_items = sorted(d.items(), key=lambda x: x[1], reverse=True)

# top 3 items print karna
print(sorted_items[:3])
📊 Output:

[('b', 50), ('d', 40), ('c', 30)]
