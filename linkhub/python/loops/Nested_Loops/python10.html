# Question 10:
# Use break in outer loop and print numbers using nested loop.

# outer loop
for i in range(5):

    # stop loop when i == 3
    if i == 3:
        break

    # inner loop
    for j in range(1,4):
        print(j, end=" ")

    # move to next line
    print()
📊 Output:

1 2 3
1 2 3
1 2 3
