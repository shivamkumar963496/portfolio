# Question 22:
# Price aur discount % input lo aur final price print karo

# user se price aur discount input le rahe hain
price = float(input("ENTER PRICE = "))
discount = float(input("ENTER DISCOUNT % = "))

# final price calculate kar rahe hain
final = price - (price * discount / 100)

# result print kar rahe hain
print("FINAL PRICE AFTER DISCOUNT =", final)
