# Question 5:
# Dictionary me value update karo

# dictionary create karna
d = {"name":"shivam","father":"yashpal","city":"muzaffarnagar"}

# old dictionary print karna
print("old dic :", d)

# value update karna
d["city"] = "meerut"

# new dictionary print karna
print("New dic :", d)
📊 Output:

old dic : {'name': 'shivam', 'father': 'yashpal', 'city': 'muzaffarnagar'}
New dic : {'name': 'shivam', 'father': 'yashpal', 'city': 'meerut'}
