# Question 23:
# Ek Python program likho jo tuple me unique elements ki count nikaale.

# tuple create karna
t = (22, 36, 98, 75, 49, 86, 91, 43, 25, 77, 89, 22, 89, 75, 36)

# unique elements ki count nikalna
print("Tuple me unique elements ki count :", len(set(t)))
📊 Output:

Tuple me unique elements ki count : 11
