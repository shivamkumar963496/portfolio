# Question 18:
# List comprehension ka use karke
# do lists ka Cartesian Product banao.

# First list create karna
a = [1, 2]

# Second list create karna
b = ["a", "b"]

# Cartesian Product banana
result = [(x, y) for x in a for y in b]

# Final output print karna
print(result)
📊 Output:

[(1, 'a'), (1, 'b'), (2, 'a'), (2, 'b')]
