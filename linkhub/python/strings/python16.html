# Question 16:
# Count uppercase and lowercase characters in a string.

# input from user
s = input("enter sentence : ")

# initialize counters
upper = 0
lower = 0

# loop through string
for ch in s:
    if ch.isupper():
        upper += 1
    elif ch.islower():
        lower += 1

# print result
print("Uppercase:", upper)
print("Lowercase:", lower)
📊 Output (Example):

Input: Hello World
Uppercase: 2
Lowercase: 8
