# Question 1:
# List ka sum nikalna hai aur total elements count karne hain

# List create ki
nums = [2,1,4,8,9,7,6,5,4,3,8]

# Variables initialize kiye
total = 0
count = 0

# Loop chala kar sum aur count nikala
for i in nums:
    total += i      # har element add ho raha hai
    count += 1      # elements count ho rahe hain

# Output print kiya
print("The sum of list :", total)
print("Total number of elements :", count)
📊 Output:

The sum of list : 57
Total number of elements : 11
