# Question 9:
# Ek Python program likho jo dictionary ki sabhi values ko list me convert kare.

# dictionary create karna
d = {
    'shivam': 23,
    'yograj': 34,
    'akhand': 21,
    'rajveer': 23,
    'atul': 54
}

# values ko list me convert karna
values_list = list(d.values())

# output print karna
print(values_list)
📊 Output:

[23, 34, 21, 23, 54]
