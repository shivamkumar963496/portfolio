# Question 16:
# Ek Python program likho jo dictionary ko values ke basis par sort kare.

# dictionary create karna
d = {
    "a": 3,
    "b": 1,
    "c": 2
}

# values ke basis par sort karna
sorted_d = dict(sorted(d.items(), key=lambda item: item[1]))

# output print karna
print(sorted_d)
📊 Output:

{'b': 1, 'c': 2, 'a': 3}
