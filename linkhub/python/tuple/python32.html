# Ek Python program likho jo tuple ka second smallest element find kare.

# tuple create karna
t = (25, 65, 47, 89, 65, 18, 28, 96, 35, 89, 99, 74, 58, 29, 94)

# second smallest element find karna
second_small = sorted(set(t))[1]

# output print karna
print("Second smallest element :", second_small)
📊 Output:

Second smallest element : 25
