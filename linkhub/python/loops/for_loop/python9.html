# Question 9:
# Write a Python program to take a string as input
# and count total number of vowels in it.

# user se string input liya ja raha hai
s = input("ENTER STRING : ")

# vowel count karne ke liye variable
count = 0

# string ke har character ko check kiya
for i in s:
    if i in "aeiouAEIOU":
        count += 1

# total vowels print kiya
print("VOWELS =", count)
