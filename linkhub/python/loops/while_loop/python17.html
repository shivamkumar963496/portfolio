# Question 17:
# Count occurrences of a digit in a number.

# input lena
n = int(input("Enter a number: "))
d = int(input("Enter a digit: "))

# count start
count = 0

# digits check
while n != 0:
    last_digit = n % 10
    if last_digit == d:
        count += 1
    n //= 10

# result print
print(f"Digit {d} appears {count} times")
📊 Output:

Enter a number: 122333
Enter a digit: 3
Digit 3 appears 3 times
