# Question 10:
# Find factorial of a number.

# input lena
n = int(input("Enter a number: "))

# starting values
i = 1
fact = 1

# 1 se n tak multiply
while i <= n:
    fact *= i   # factorial update
    i += 1      # next number

# result print
print("Factorial of the number =", fact)
📊 Output:

Enter a number: 5
Factorial of the number = 120
