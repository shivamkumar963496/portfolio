# Question 5:
# Print reverse number triangle pattern using nested loop.

# total rows
n = 5

# outer loop (reverse)
for i in range(n, 0, -1):

    # inner loop for numbers (reverse)
    for j in range(i, 0, -1):
        print(j, end=" ")
        
    # move to next line
    print()
📊 Output:

5 4 3 2 1
4 3 2 1
3 2 1
2 1
1
