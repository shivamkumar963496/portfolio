# Question 31:
# Nested dictionary se specific values access karo

# dictionary create karna
students={
    "amit":{"marks":90,"age":21},
    "shivam":{"marks":95,"age":22}
}

# output print karna
print("amit ke marks : ", students["amit"]["marks"])
print("shivam ki age : ", students["shivam"]["age"])
📊 Output:

amit ke marks : 90
shivam ki age : 22
