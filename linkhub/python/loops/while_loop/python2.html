# Question 2:
# Write a Python program to take a number as input
# and print numbers from n to 1 using while loop (reverse order).
# (Ek number input lo aur n se 1 tak reverse me print karo)

# user se number input liya ja raha hai
n = int(input("Enter number: "))

# starting value n se li gayi hai
i = n

# jab tak i 1 se bada ya barabar hai tab tak loop chalega
while i >= 1:
    print(i, end=" ")
    i -= 1
📊 Output:

Enter number: 5
5 4 3 2 1
