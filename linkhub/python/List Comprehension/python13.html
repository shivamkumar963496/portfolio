# Question 13:
# List comprehension ka use karke
# 1 se 100 tak jo numbers
# 3 aur 5 dono se divide hote hain
# unko find karo.

# 1 se 100 tak numbers create karna
nums = list(range(1, 101))

# 3 aur 5 dono se divisible numbers find karna
result = [x for x in nums if x % 3 == 0 and x % 5 == 0]

# Final output print karna
print(result)
📊 Output:

[15, 30, 45, 60, 75, 90]
