# Question 28:
# Dictionary ki values ko string me convert karke update karo

# dictionary create karna
d = {"a": 2, "b": 4, "c": 3}

# loop ke through values ko string me convert karna
for k in d:
    d[k] = str(d[k])

# output print karna
print(d)
📊 Output:

{'a': '2', 'b': '4', 'c': '3'}
