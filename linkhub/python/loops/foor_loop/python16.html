# Question 16:
# Write a Python program to check whether a number is Armstrong or not.

# user se number input liya ja raha hai
n = int(input("ENTER A NUMBER : "))

# original number ko store kiya
temp = n

# sum store karne ke liye variable
total = 0

# har digit ka cube add kiya
while temp > 0:
    digit = temp % 10
    total += digit**3
    temp //= 10

# Armstrong condition check ki
if total == n:
    print("ARMSTRONG NUMBER")
else:
    print("NOT ARMSTRONG")
