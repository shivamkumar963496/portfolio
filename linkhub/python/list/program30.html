# Question 30:
# Words list ko print karna hai aur uski sorted unique list banani hai

# List create ki
words = ['a','b','c','a','c','d','e']

# Original list print ki
print("Normal list :", words)

# set se duplicates remove kiye aur sorted se arrange kiya
print("Sorted and unique list :", sorted(set(words)))
📊 Output:

Normal list : ['a', 'b', 'c', 'a', 'c', 'd', 'e']
Sorted and unique list : ['a', 'b', 'c', 'd', 'e']
