# Question 3:
# Print a number pattern from 1 to 5, skipping the number 2 using continue.

# outer loop for rows
for i in range(1,6):

    # inner loop for numbers
    for j in range(1, i+1):

        # skip number 2
        if j == 2:
            continue

        print(j, end=" ")
        
    # move to next line
    print()
📊 Output:

1
1
1 3
1 3 4
1 3 4 5
