# Question 18:
# Calculate power of a number (base^power).

# input lena
base = int(input("Enter base: "))
power = int(input("Enter power: "))

# original power save
temp = power

# result start
result = 1

# multiply base, power times
while power > 0:
    result *= base
    power -= 1

# result print
print(f"{base}^{{temp}} =", result)
📊 Output:

Enter base: 2
Enter power: 3
2^3 = 8
