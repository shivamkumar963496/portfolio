# Question 36:
# Ek Python program likho jo tuple ke positive aur negative elements alag-alag tuples me store kare.

# tuple create karna
t = (-5, 1, 4, 5, 6, -7, -4, -2, -8, 6, 4, 8, 7)

# positive elements nikalna
positive_tuple = tuple(i for i in t if i > 0)

# negative elements nikalna
negative_tuple = tuple(i for i in t if i < 0)

# output print karna
print("Mix tuple :", t)
print("Positive elements tuple :", positive_tuple)
print("Negative elements tuple :", negative_tuple)
📊 Output:

Mix tuple : (-5, 1, 4, 5, 6, -7, -4, -2, -8, 6, 4, 8, 7)
Positive elements tuple : (1, 4, 5, 6, 6, 4, 8, 7)
Negative elements tuple : (-5, -7, -4, -2, -8)
