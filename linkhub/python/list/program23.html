# Question 23:
# List me odd numbers ki count nikalni hai (generator expression ka use karke)

# List create ki
nums = [10,25,5,40,12,36,98,75,45,68,92,91]

# Generator expression se odd count nikala
odd_digits = sum(1 for i in nums if i % 2 != 0)

# Output print kiya
print("Odd Count:", odd_digits)
📊 Output:

Odd Count : 5
