# Question 20:
# Ek Python program likho jo user se do sets lekar unka intersection nikaale.

# user se input lena
a = set(map(int, input("Enter 1st set : ").split()))
b = set(map(int, input("Enter 2nd set : ").split()))

# intersection nikalna
print(a & b)
📊 Output (example):

Enter 1st set : 1 2 3 4
Enter 2nd set : 3 4 5 6
{3, 4}
