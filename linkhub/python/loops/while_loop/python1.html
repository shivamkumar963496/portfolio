# Question 1:
# Write a Python program to take a number as input
# and print numbers from 1 to n using while loop.
# (Ek number input lo aur 1 se n tak print karo)

# user se number input liya ja raha hai
n = int(input("Enter number: "))

# starting value 1 se li gayi hai
i = 1

# jab tak i n se chhota ya barabar hai tab tak loop chalega
while i <= n:+
    print(i, end=" ")
    i += 1
📊 Output:

Enter number: 5
1 2 3 4 5
