# Question 6:
# Ek Python program likho jo dictionary me kisi key ki value update kare.

# dictionary create karna
d = {"a": 10, "b": 20, "c": 30}

# old dictionary print karna
print("Old dictionary :", d)

# key aur new value define karna
key = "b"
new_value = 50

# value update karna
d[key] = new_value

# new dictionary print karna
print("New dictionary :", d)
📊 Output:

Old dictionary : {'a': 10, 'b': 20, 'c': 30}
New dictionary : {'a': 10, 'b': 50, 'c': 30}
