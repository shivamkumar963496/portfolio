# Question 24:
# List comprehension ka use karke
# matrix me se even numbers find karo.

# Matrix create karna
matrix = [
    [1, 2],
    [3, 4]
]

# Even numbers extract karna
result = [
    num
    for row in matrix
    for num in row
    if num % 2 == 0
]

# Final output print karna
print(result)
📊 Output:

[2, 4]
