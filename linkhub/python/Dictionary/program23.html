# Question 23:
# Dictionary ki sabhi values ka multiplication nikaalo

# dictionary create karna
d = {"a": 2, "b": 3, "c": 4}

# initial result set karna
result = 1

# values ka multiplication karna
for v in d.values():
    result *= v

# output print karna
print(result)
📊 Output:

24
