# Question 15:
# List ke elements ka frequency count karo

# list create karna
lst = [1, 2, 2, 3]

# empty dictionary banana
freq = {}

# loop ke through frequency count karna
for i in lst:
    freq[i] = freq.get(i, 0) + 1

# output print karna
print(freq)
📊 Output:

{1: 1, 2: 2, 3: 1}
