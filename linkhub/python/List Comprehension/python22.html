# Question 22:
# List comprehension ka use karke
# matrix ka transpose find karo.

# Matrix create karna
matrix = [
    [1, 2, 3],
    [4, 5, 6]
]

# Matrix transpose karna
result = [
    [row[i] for row in matrix]
    for i in range(len(matrix[0]))
]

# Final output print karna
print(result)
📊 Output:

[[1, 4], [2, 5], [3, 6]]
