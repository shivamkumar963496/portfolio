# Question 25:
# Dictionary comprehension ka use karke
# keys aur values ko dictionary me convert karo.

# Keys list create karna
keys = ["a", "b"]

# Values list create karna
values = [1, 2]

# Dictionary create karna
result = {k: v for k, v in zip(keys, values)}

# Final output print karna
print(result)
📊 Output:

{'a': 1, 'b': 2}
