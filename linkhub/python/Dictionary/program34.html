# Question 34:
# 0 se 9 tak ke even numbers ko dictionary me store karo

# empty dictionary
dic = {}

# loop ke through even numbers add karna
for i in range(0, 10):
    if i % 2 == 0:
        dic[i] = i

# output print karna
print(dic)
📊 Output:

{0: 0, 2: 2, 4: 4, 6: 6, 8: 8}
