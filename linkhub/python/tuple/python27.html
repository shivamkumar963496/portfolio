# Question 27:
# Ek Python program likho jo nested tuple ko flatten kare.

# nested tuple create karna
t = ((1, 2, 3), (4, 5, 6), (7, 8, 9))

# empty tuple banana
flat = ()

# nested loop se flatten karna
for i in t:
    for j in i:
        flat += (j,)

# output print karna
print(flat)
📊 Output:

(1, 2, 3, 4, 5, 6, 7, 8, 9)
