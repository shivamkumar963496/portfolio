# Question 19:
# Ek Python program likho jo tuple ko reverse kare using slicing.

# tuple create karna
t = (22, 36, 98, 75, 49, 86, 91, 43, 25, 77, 89)

# tuple reverse karna
rev = t[::-1]

# output print karna
print("Reverse tuple :", rev)
📊 Output:

Reverse tuple : (89, 77, 25, 43, 91, 86, 49, 75, 98, 36, 22)
