# Question 12:
# Dictionary me max value aur max key find karo

# dictionary create karna
d = {"a": 5, "b": 50}

# max value nikalna
print("dictionary ki sabse badi value :", max(d.values()))

# max key nikalna
print("dictionary ki sabse badi key :", max(d.keys()))
📊 Output:

dictionary ki sabse badi value : 50
dictionary ki sabse badi key : b
