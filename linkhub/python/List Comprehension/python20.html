# Question 20:
# List comprehension ka use karke
# values ko Boolean form me convert karo.

# Mixed value list create karna
nums = [0, 1, "", "hi"]

# Boolean conversion karna
result = [bool(x) for x in nums]

# Final output print karna
print(result)
📊 Output:

[False, True, False, True]
