# Question 35:
# List ka minimum aur 2nd largest element find karna hai

# List create ki
nums = [1,2,2,3,3,3]

# Minimum element nikala
print("List ka sabse chota digit :", min(nums)) 

# 2nd largest (duplicate safe) nikala
print("\nList ka 2nd largest number :", sorted(set(nums))[-2])
📊 Output:

List ka sabse chota digit : 1
List ka 2nd largest number : 2
