# Question 25:
# Find HCF (GCD) of two numbers.

# input lena
a = int(input("Enter first number: "))
b = int(input("Enter second number: "))

# Euclid method (subtraction)
while a != b:
    if a > b:
        a = a - b
    else:
        b = b - a

# result print
print("HCF =", a)
📊 Output:

Enter first number: 12
Enter second number: 18
HCF = 6
