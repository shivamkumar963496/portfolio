# Question 12:
# Ek Python program likho jo sentence ke words ki frequency count kare.

# string create karna
s = "my name is shivam kumar i love python i am learning python"

# words me split karna
words = s.split()

# empty dictionary banana
freq = {}

# word frequency count karna
for w in words:
    if w in freq:
        freq[w] += 1
    else:
        freq[w] = 1

# output print karna
print(freq)
📊 Output:

{'my': 1, 'name': 1, 'is': 1, 'shivam': 1, 'kumar': 1, 'i': 3, 'love': 1, 'python': 2, 'am': 1, 'learning': 1}
