# Question 5:
# Find sum of numbers from 1 to n.

# input lena
n = int(input("Enter number: "))

# starting values
i = 1
sum = 0

# loop 1 se n tak
while i <= n:
    sum += i   # sum me add
    i += 1     # next number

# final result
print("Sum =", sum)
📊 Output:

Enter number: 5
Sum = 15
