# Question 19:
# Dictionary ko reverse karo (value → key)

# dictionary create karna
d = {"a": 1, "b": 2, "c": 3, "d": 4, "e": 5}

# empty dictionary banana
rev = {}

# reverse mapping karna
for k, v in d.items():
    rev[v] = k

# output print karna
print(rev)
📊 Output:

{1: 'a', 2: 'b', 3: 'c', 4: 'd', 5: 'e'}
