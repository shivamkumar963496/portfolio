# Question 19:
# List comprehension ka use karke
# duplicate numbers remove karo.

# Number list create karna
nums = [1, 2, 2, 3, 1]

# Seen list create karna
seen = []

# Duplicate remove karna
result = [x for x in nums if not (x in seen or seen.append(x))]

# Final output print karna
print(result)
📊 Output:

[1, 2, 3]
