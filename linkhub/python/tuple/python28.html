# Question 28:
# Ek Python program likho jo tuple ke elements ki frequency dictionary me store kare.

# tuple create karna
t = (1, 5, 8, 9, 7, 4, 5, 5, 6, 2, 8, 9, 7, 5, 2, 1, 4, 3, 9, 5, 2)

# empty dictionary banana
freq = {}

# frequency count karna
for i in t:
    freq[i] = t.count(i)

# output print karna
print(freq)
📊 Output:

{1: 2, 5: 5, 8: 2, 9: 3, 7: 2, 4: 2, 6: 1, 2: 3, 3: 1}
