# Question 2:
# Ek Python program likho jo dictionary se given key ka value print kare.

# dictionary create karna
d = {'shivam': 23, 'yograj': 34, 'akhand': 21, 'rajveer': 23, 'atul': 54}

# key define karna
key = "shivam"

# value access karna
print(d[key])
📊 Output:

23
