# Question 28:
# List me total duplicate elements ki count nikalni hai

# List create ki
nums = [10,25,5,40,12,36,98,75,45,68,92,91,5,25,10,98]

# Total elements - unique elements = duplicate count
print("Duplicate count :", len(nums) - len(set(nums)))
📊 Output:

Duplicate count : 4
