# Question 19:
# Ek Python program likho jo dictionary se specific value wale elements remove kare.

# dictionary create karna
d = {"a": 10, "b": 20, "c": 10}

# target value define karna
target = 10

# new dictionary banana (filtering)
new_dic = {k: v for k, v in d.items() if v != target}

# output print karna
print(new_dic)
📊 Output:

{'b': 20}
