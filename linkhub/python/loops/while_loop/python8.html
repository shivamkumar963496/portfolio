# Question 8:
# Reverse a number.

# input lena
n = int(input("Enter a number: "))

# reverse store karne ke liye
reverse = 0

# jab tak number 0 nahi hota
while n != 0:
    last_digit = n % 10        # last digit nikalo
    reverse = reverse * 10 + last_digit  # reverse banao
    n //= 10                   # last digit hatao

# result print
print("Reverse number =", reverse)
📊 Output:

Enter a number: 1234
Reverse number = 4321
