# Question 9:
# List comprehension ka use karke
# string numbers ko integer me convert karo.

# String number list create karna
nums = ["1", "2", "3"]

# String ko integer me convert karna
result = [int(x) for x in nums]

# Final output print karna
print(result)
📊 Output:

[1, 2, 3]
