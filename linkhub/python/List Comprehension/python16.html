# Question 16:
# List comprehension ka use karke
# number aur uska square tuple me store karo.

# Number list create karna
nums = [1, 2, 3, 4, 5, 6, 7, 8, 9]

# Number aur square tuple banana
result = [(x, x * x) for x in nums]

# Final output print karna
print(result)
📊 Output:

[(1, 1), (2, 4), (3, 9), (4, 16), (5, 25), (6, 36), (7, 49), (8, 64), (9, 81)]
