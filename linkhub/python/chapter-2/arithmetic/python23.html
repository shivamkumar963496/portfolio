# Question 23:
# 3 digit number input lo aur uske digits ka sum print karo

# user se number input le rahe hain
n = int(input("ENTER 3 DIGIT NUMBER = "))

# digits nikal kar sum calculate kar rahe hain
d1 = n % 10
d2 = (n // 10) % 10
d3 = n // 100

total = d1 + d2 + d3

# result print kar rahe hain
print("THE SUM OF DIGITS =", total)
