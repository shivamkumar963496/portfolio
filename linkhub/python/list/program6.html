# Question 6:
# List me maximum element find karna hai

# List create ki
nums = [10,25,5,40,12,36,98,75,45,68,92,91]

# Pehla element ko max maan liya
m = nums[0]

# Loop chala kar maximum find kiya
for i in nums:
    if i > m:       # agar current element bada hai
        m = i       # to update kar do

# Output print kiya
print("The maximum element of the list :", m)
📊 Output:

The maximum element of the list : 98
