# Question 6:
# List comprehension ka use karke
# har number ko 2 se multiply karo.

# Number list create karna
nums = [1, 2, 3, 4, 5, 6, 7, 8, 9]

# Har number ko 2 se multiply karna
result = [i * 2 for i in nums]

# Final output print karna
print(result)
📊 Output:

[2, 4, 6, 8, 10, 12, 14, 16, 18]
