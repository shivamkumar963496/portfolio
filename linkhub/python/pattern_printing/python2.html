# Question 2:
# Print right triangle star pattern using nested loop.

# outer loop for rows
for i in range(1,6):

    # inner loop for stars
    for j in range(1, i+1):
        print("*", end=" ")
        
    # move to next line
    print()
📊 Output:

*
* *
* * *
* * * *
* * * * *
