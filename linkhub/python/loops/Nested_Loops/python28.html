# Question 28:
# Print pairs (i,j) where sum is odd using continue.

# lists
a = [1,2,3,4]
b = [9,8,7,6]

# nested loop
for i in a:
    for j in b:

        # skip even sum
        if (i + j) % 2 == 0:
            continue

        print(i, j)
📊 Output:

1 8
1 6
2 9
2 7
3 8
3 6
4 9
4 7
