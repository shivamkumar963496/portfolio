# Question 35:
# Ek Python program likho jo tuple ke har element ka square nikaal kar naya tuple banaye.

# tuple create karna
t = (25, 65, 47, 89, 65, 18, 28, 96, 35, 89, 99)

# square tuple banana
square_tuple = tuple(i * i for i in t)

# output print karna
print("Normal tuple :", t)
print("Square tuple :", square_tuple)
📊 Output:

Normal tuple : (25, 65, 47, 89, 65, 18, 28, 96, 35, 89, 99)
Square tuple : (625, 4225, 2209, 7921, 4225, 324, 784, 9216, 1225, 7921, 9801)
