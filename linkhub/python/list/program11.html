# Question 11:
# List me minimum element find karna hai

# List create ki
nums = [10,25,5,40,12,36,98,75,45,68,92,91]

# Pehla element ko minimum maan liya
m = nums[0]

# Loop chala kar minimum find kiya
for i in nums:
    if i < m:      # agar current element chhota hai
        m = i      # to update kar do

# Output print kiya
print("Minimum element in the list :", m)
📊 Output:

Minimum element in the list : 5
