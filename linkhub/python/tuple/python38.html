# Ek Python program likho jo do tuples ko zip karke pairs banaye.

# tuples create karna
t1 = (1, 2, 3)
t2 = (4, 5, 6)

# zip karke pairs banana
zipped = tuple(zip(t1, t2))

# output print karna
print("Tuples ko zip karke pair :", zipped)
📊 Output:

Tuples ko zip karke pair : ((1, 4), (2, 5), (3, 6))
