# Question 1:
# Do lists ko zip karke dictionary banao.

# lists create karna
name = ["shivam","rohit","shubham","yograj"]
age = [25,21,24,27]

# dictionary banana
d = dict(zip(name, age))

# output print karna
print(d)
📊 Output:

{'shivam': 25, 'rohit': 21, 'shubham': 24, 'yograj': 27}
