# Question 25:
# Ek Python program likho jo dictionary ki duplicate values find kare.

# dictionary create karna
d = {"a": 1, "b": 2, "c": 1}

# values list me convert karna
values = list(d.values())

# duplicate values nikalna
duplicate = [v for v in set(values) if values.count(v) > 1]

# output print karna
print(duplicate)
📊 Output:

[1]
