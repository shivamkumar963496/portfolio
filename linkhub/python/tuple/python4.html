# Question 4:
# Ek Python program likho jo tuple ke last aur second last element ko access kare.

# tuple create karna
t = (10, 20, 30, 40, 50)

# last element access karna
print("Last element :", t[-1])

# second last element access karna
print("Second last element :", t[-2])
📊 Output:

Last element : 50
Second last element : 40
