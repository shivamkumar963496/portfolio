# Question 23:
# List comprehension ka use karke
# string ke sabhi substrings generate karo.

# String create karna
s = "abc"

# All substrings generate karna
result = [
    s[i:j]
    for i in range(len(s))
    for j in range(i + 1, len(s) + 1)
]

# Final output print karna
print(result)
📊 Output:

['a', 'ab', 'abc', 'b', 'bc', 'c']
