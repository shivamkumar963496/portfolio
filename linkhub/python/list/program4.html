# Question 4:
# User se list input lekar sirf even numbers ka sum nikalna hai

# User input liya aur list me convert kiya
nums = list(map(int, input("enter list elements : ").split()))

# total variable initialize kiya
total = 0

# Loop chala kar even numbers ka sum nikala
for i in nums:
    if i % 2 == 0:     # check even number
        total += i     # even number add ho raha hai

# Output print kiya
print("Even Sum =", total)
📊 Output Example:

Input: 1 2 3 4 5 6
Even Sum = 12
