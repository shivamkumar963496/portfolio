# Question 26:
# Dictionary ki values ka square karke new dictionary me store karo

# dictionary create karna
d = {"a": 2, "b": 4, "c": 3}

# new dictionary banana
result = {}

# loop ke through square nikalna
for k, v in d.items():
    result[k] = v ** 2

# output print karna
print(result)
📊 Output:

{'a': 4, 'b': 16, 'c': 9}
