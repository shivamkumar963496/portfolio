# Question 13:
# Print binary triangle pattern using nested loop.

# outer loop
for i in range(1,5):

    # inner loop
    for j in range(1, i+1):

        # condition for 1 and 0
        if (i + j) % 2 == 0:
            print("1", end=" ")
        else:
            print("0", end=" ")

    # move to next line
    print()
📊 Output:

1
0 1
1 0 1
0 1 0 1
