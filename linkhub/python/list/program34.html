# Question 34:
# User se N le kar list ka Nth largest element find karna hai

# List create ki
nums = [10,25,5,40,12,36,98,75,45,68,92,91,5,25,10,98]

# User se N input liya
n = int(input("Enter Nth largest number : "))

# Check kiya ki N valid hai ya nahi
if n < len(nums):
    print(sorted(nums)[-n])   # Nth largest element
else:
    print("Invalid nth number")
📊 Output Example:

Input: 2
Output: 98
