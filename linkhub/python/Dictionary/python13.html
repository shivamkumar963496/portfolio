# Question 13:
# Ek Python program likho jo dictionary ke keys aur values ko swap kare.

# dictionary create karna
d = {'a': 1, 'b': 2, 'c': 3, 'd': 4}

# keys aur values swap karna
new_d = {v: k for k, v in d.items()}

# output print karna
print(new_d)
📊 Output:

{1: 'a', 2: 'b', 3: 'c', 4: 'd'}
