# Question 15:
# Print triangle star pattern but skip center position in each row.

# total rows
n = 5

# outer loop
for i in range(1, n+1):

    # inner loop
    for j in range(1, i+1):

        # find center position
        center = (i + 1) // 2

        # skip center star
        if j == center:
            continue

        print("*", end="")

    # move to next line
    print()
📊 Output:


*
**
***
****
