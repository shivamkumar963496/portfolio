# Question 3:
# Dictionary se value safely access karo using get()

# dictionary create karna
d = {"name": "Amit", "age": 20, "city": "muzaffarnagar"}

# existing key access karna
print("city :", d.get("city","not found"))

# non-existing key access karna
print("father name :", d.get("father name","not found"))
📊 Output:

city : muzaffarnagar
father name : not found
