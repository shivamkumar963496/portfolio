# Question 21:
# Ek Python program likho jo tuple ke odd aur even elements alag-alag tuples me store kare.

# tuple create karna
t = (22, 36, 98, 75, 49, 86, 91, 43, 25, 77, 89)

# odd elements nikalna
odd_tuple = tuple(i for i in t if i % 2 != 0)

# even elements nikalna
even_tuple = tuple(i for i in t if i % 2 == 0)

# output print karna
print("Tuple ke odd elements :", odd_tuple)
print("Tuple ke even elements :", even_tuple)
📊 Output:

Tuple ke odd elements : (75, 49, 91, 43, 25, 77, 89)
Tuple ke even elements : (22, 36, 98, 86)
