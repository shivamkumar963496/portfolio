# Question 10:
# Dictionary ke key-value pairs ko loop se print karo

# dictionary create karna
d = {"name": "Amit", "age": 20, "city": "muzaffarnagar"}

# loop ke through print karna
for k, v in d.items():
    print(k + ":" + str(v))
📊 Output:

name:Amit
age:20
city:muzaffarnagar
