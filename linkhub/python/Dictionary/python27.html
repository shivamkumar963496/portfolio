# Question 27:
# Ek Python program likho jo nested dictionary ko flatten kare.

# nested dictionary create karna
d = {"a": {"b": 1}}

# empty dictionary banana
flat = {}

# flatten logic
for k, v in d.items():
    for k2, v2 in v.items():
        flat[k + "." + k2] = v2

# output print karna
print(flat)
📊 Output:

{'a.b': 1}
