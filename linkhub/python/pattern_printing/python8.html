# Question 8:
# Print right aligned pyramid star pattern using nested loop.

# outer loop for rows
for i in range(1,5):

    # space loop
    for j in range(4 - i):
        print(" ", end="")

    # star loop
    for k in range(i):
        print("*", end=" ")

    # move to next line
    print()
📊 Output:

*
* *
* * *
* * * *
