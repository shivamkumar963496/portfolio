# Question 7:
# List ko sort karke second maximum element find karna hai

# List create ki
nums = [10,25,5,40,12,36,98,75,45,68,92,91]

# List ko sort kiya
nums.sort()

# Sorted list print ki
print("\nSorted list :", nums)

# Second last element (second maximum) nikala
print("\nThe 2nd maximum element of the list :", nums[-2])
📊 Output:

Sorted list : [5, 10, 12, 25, 36, 40, 45, 68, 75, 91, 92, 98]
The 2nd maximum element of the list : 92
