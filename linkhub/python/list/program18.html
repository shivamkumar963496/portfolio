# Question 18:
# List ko reverse karna hai (index loop ka use karke)

# List create ki
nums = [10,25,5,40,12,36,98,75,45,68,92,91]

# Empty list banayi reverse ke liye
rev = []

# Reverse index se loop chala
for i in range(len(nums)-1, -1, -1):
    rev.append(nums[i])   # last se elements add ho rahe hain

# Output print kiya
print(rev)
📊 Output:

[91, 92, 68, 45, 75, 98, 36, 12, 40, 5, 25, 10]
