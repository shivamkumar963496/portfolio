# Question 14:
# Print alphabet triangle pattern using nested loop.

# outer loop
for i in range(1,5):

    # starting ASCII value (A = 65)
    ch = 65

    # inner loop
    for j in range(i):
        print(chr(ch), end=" ")
        ch += 1

    # move to next line
    print()
📊 Output:

A
A B
A B C
A B C D
