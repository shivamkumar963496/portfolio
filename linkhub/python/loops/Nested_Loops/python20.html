# Question 20:
# Print rows of matrix until row sum becomes greater than 10.

# matrix
matrix = [
    [2,3,5],
    [2,3,1],
    [7,9,8]
]

# loop through rows
for row in matrix:

    # check sum of row
    if sum(row) > 10:
        break

    print(row)
📊 Output:

[2, 3, 5]
[2, 3, 1]
