# Question 15:
# Ek Python program likho jo user se element lekar uski index value tuple me find kare.

# tuple create karna
t = (22, 36, 98, 75, 49, 86, 91, 43, 25, 77, 89)

# tuple print karna
print(t)

# user se input lena
n = int(input("Kis element ki index value nikalni hai: "))

# index find karna
print(f"{n} ki index value :", t.index(n))
📊 Output:

(22, 36, 98, 75, 49, 86, 91, 43, 25, 77, 89)
Kis element ki index value nikalni hai: 75
75 ki index value : 3
