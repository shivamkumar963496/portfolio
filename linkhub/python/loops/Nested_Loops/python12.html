# Question 12:
# Print multiplication values but skip those divisible by 5.

# outer loop
for i in range(1,6):

    # inner loop
    for j in range(1,6):

        # skip multiples of 5
        if (i * j) % 5 == 0:
            continue

        print(i * j, end=" ")

    # move to next line
    print()
📊 Output:

1 2 3 4
2 4 6 8
3 6 9 12
4 8 12 16
