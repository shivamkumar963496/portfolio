# Question 15:
# Ek Python program likho jo dictionary ko keys ke basis par sort kare.

# dictionary create karna
d = {"b": 2, "a": 1, "c": 3}

# dictionary sort karna
new_d = dict(sorted(d.items()))

# output print karna
print(new_d)
📊 Output:

{'a': 1, 'b': 2, 'c': 3}
