# Question 18:
# Ek Python program likho jo list ke elements ki frequency count kare.

# list create karna
nums = [1, 2, 2, 3, 1]

# empty dictionary banana
freq = {}

# frequency count karna
for n in nums:
    freq[n] = freq.get(n, 0) + 1

# output print karna
print(freq)
📊 Output:

{1: 2, 2: 2, 3: 1}
