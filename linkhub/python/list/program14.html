# Question 14:
# User se list input lekar sabse chota aur sabse bada element find karna hai

# User input liya aur list me convert kiya
nums = list(map(int, input("ENTER LIST ELEMENTS : ").split()))

# min aur max function ka use kiya
print("Sabse chota element :", min(nums))
print("Sabse bada element :", max(nums))
📊 Output Example:

Input: 3 7 1 9 2
Sabse chota element : 1
Sabse bada element : 9
