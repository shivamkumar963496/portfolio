# Question 8:
# Negative numbers wali list me sabse bada (maximum) element find karna hai

# List create ki (all negative numbers)
nums = [-10,-2,-30,-69,-58,-47,-25,-31,-999]

# max function se maximum value nikali
print("The maximum negative element of the list :", max(nums))
📊 Output:

The maximum negative element of the list : -2
