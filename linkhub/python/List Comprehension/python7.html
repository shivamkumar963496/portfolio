# Question 7:
# List comprehension ka use karke
# 10 se bade numbers filter karo.

# Number list create karna
nums = [5, 12, 7, 18, 4, 7, 8, 1, 45, 69, 21, 10, 11, 14, 78, 65, 22, 12]

# 10 se bade numbers filter karna
result = [x for x in nums if x > 10]

# Final output print karna
print(result)
📊 Output:

[12, 18, 45, 69, 21, 11, 14, 78, 65, 22, 12]
