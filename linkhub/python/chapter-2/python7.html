📘 Python String Methods (Hindi Notes)

len()

string की length (total characters) बताता है

upper()

string को CAPITAL letters में बदलता है

lower()

string को small letters में बदलता है

strip()

left + right से extra spaces / characters हटाता है

lstrip()

left side से extra spaces / characters हटाता है

rstrip()

right side से extra spaces / characters हटाता है

replace()

एक word / character को दूसरे से बदलता है

split()

string को तोड़कर list बनाता है

join()

list को जोड़कर string बनाता है

count()

बताता है कोई word / character कितनी बार आया है

find()

word का index देता है (न मिले तो -1)

capitalize()

पहला letter CAPITAL, बाकी small करता है

startswith()

check करता है string किस word से start हो रही है

endswith()

check करता है string किस word से end हो रही है

isalpha()

check करता है string में सिर्फ alphabets हैं या नहीं

isdigit()

check करता है string में सिर्फ digits हैं या नहीं

isalnum()

check करता है string में alphabets + digits हैं या नहीं

center()

string को दिए गए width में center align करता है
