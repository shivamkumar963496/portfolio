# Question 32:
# Nested dictionary ki sabhi values ka sum nikalo

# dictionary create karna
d = {
    "a": {"x": 1, "y": 2},
    "b": {"z": 3}
}

# total variable
total = 0

# nested loop ke through sum nikalna
for inner_dict in d.values():
    for value in inner_dict.values():
        total += value

# output print karna
print(total)
📊 Output:

6
